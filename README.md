# testrail

Run [TestRail](http://www.gurock.com/testrail/) in a Docker container, for testing and development.

## Build the Image

    docker build -t testrail .

## Run the Container

    ./start_container.sh

## Usage

Browse to:

    http://localhost:7070/testrail/
